# 🔊 적 레이저 사운드 파일

이 디렉토리에 적 레이저 관련 사운드 파일을 추가해주세요.

## 필요한 사운드 파일

### 1. enemy_laser_charge.mp3
- **설명**: 적이 레이저를 충전할 때 재생되는 사운드
- **길이**: 약 5초 (충전 시간에 맞춤)
- **특징**: 점점 강해지는 에너지 충전 소리

### 2. enemy_laser_fire.mp3
- **설명**: 적이 레이저를 발사할 때 재생되는 사운드
- **길이**: 약 1초 (발사 시간에 맞춤)
- **특징**: 강력한 레이저 발사음

## 레이저 시스템 동작

1. **충전 단계** (5초)
   - `enemy_laser_charge.mp3` 재생
   - 적 위에 충전 진행도 바 표시
   - 80% 이상 충전 시 깜빡임 경고

2. **발사 단계** (1초)
   - `enemy_laser_fire.mp3` 재생
   - 빨간색 레이저 빔 발사
   - 플레이어 추적

3. **쿨다운 단계** (1초)
   - 사운드 없음
   - 다시 충전 단계로 복귀

## 사운드 파일 형식

- 포맷: MP3, WAV, OGG 등 pygame이 지원하는 형식
- 권장 비트레이트: 128kbps 이상
- 샘플레이트: 44100Hz

## 주의사항

- 사운드 파일이 없어도 게임은 정상적으로 실행됩니다.
- 파일명은 반드시 위에 명시된 이름과 정확히 일치해야 합니다.
- 파일을 추가한 후 게임을 재시작해야 사운드가 적용됩니다.
